Welcome to the Game of War!

In order to watch the game play out you must use DevTools. First click command(⌘) + option(⌥) + "j" to open the console. To restart the game just refresh the page and a new game will run, enjoy!

Although this is a simulated game where the computer plays against itself, to make things interesting pick either player 1 or player 2 to win and see if you can guess the correct winner!
